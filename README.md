# FireSafe India — Android apps

Two Android apps for fire-safety compliance, backed by a live cloud database.

## Downloads

- **FireSafe India** (for institutes, officers, contractors & auditors) →
  [firesafe-india.apk](../../releases/latest/download/firesafe-india.apk)
- **FireSafeIndians** (for citizens — see any building's live fire readiness,
  no account needed) →
  [firesafeindians.apk](../../releases/latest/download/firesafeindians.apk)

## Installing

1. On your Android phone, tap the download link above.
2. When prompted, allow **Install unknown apps** for your browser
   (Settings → Apps → your browser → Install unknown apps).
3. Open the downloaded APK and tap **Install**.

Both apps ship in **English and Hindi** and follow your device language.
